    Class HystrixBadRequestException

    An exception representing an error with provided arguments or state rather than an execution failure.

    Unlike all other exceptions thrown by a HystrixCommand<TRequest, TResult> this will not trigger fallback, not count against failure metrics and thus not trigger the circuit breaker.

    Important

    NOTE: This should only be used when an error is due to user input such as System.ArgumentException otherwise it defeats the purpose of fault-tolerance and fallback behavior.
